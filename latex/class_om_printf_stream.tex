\doxysubsection{Detailed Description}
A limited implementation of printf semantics that streams to a consumer. It doesn\textquotesingle{}t handle positional arguments... and may have other subtle omissions. But works for basic typical printf. Doesn\textquotesingle{}t depend on an in-\/memory char target big enough for the result.
