Detailed Description

template<int CLOCK_PIN, int DATA_PIN, int USE_SPI = 0>
class OmSk9822Writer< CLOCK_PIN, DATA_PIN, USE_SPI >

Create a writer for certain pins, or SPI. On the ESP8266, the SPI pins MUST be Clock D5, Data D7, and it reserves the other two SPI pins. TODO: respect the pin requests on ESP32.
